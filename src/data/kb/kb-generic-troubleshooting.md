# General IT Issue Intake

When the issue is vague ("computer broken", "nothing works"), ask clarifying questions:

1. What exactly happens? (error message, blank screen, slowness)
2. When did it start?
3. One device or all devices?
4. Any recent changes (updates, new software, travel)?

Do not guess — gather symptoms before searching KB or status.
